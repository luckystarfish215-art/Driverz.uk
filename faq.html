FAQ

Frequently asked questions

Where does fuel price data come from?

The site is structured to use official UK fuel price CSV data through the scheduled GitHub and Vercel workflow, with the live API route reading from the repository data file when available.

Do I need to share my location?

No. You can type a town, city or postcode. Location is only requested after you press the location button.

Why use a 3-mile comparison?

Three miles is a practical local default. You can widen the radius for longer journeys, but driving far to save a small amount on fuel may not be worth it.

Can I filter EV chargers by connector?

Yes. The EV search interface includes connector filtering for Type 2, CCS, CHAdeMO and Tesla-compatible options. Availability still depends on the external charger data feed.

How are cookies and analytics handled?

Essential preferences are stored locally. Google Analytics and Vercel Analytics are only loaded after a visitor accepts analytics cookies.

Can Driverz UK add more tools later?

Yes. The structure supports additional pages for traffic alerts, route planning, public APIs and more driver calculators.
